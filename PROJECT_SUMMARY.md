## 🚀 使用方法

### 直接访问
1. 打开 `index.html` 文件
2. 在浏览器中查看网站
3. 测试所有功能

### 本地服务器 (推荐)
```bash
# 使用Python
python -m http.server 8000

# 使用Node.js
node start-server.js

# 使用批处理文件 (Windows)
start-server.bat
```
